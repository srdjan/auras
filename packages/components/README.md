# @aura/components

Light-DOM interactive components for Aura.

Current exports:

- `AuraMasterDetail`
- `AuraTabs`
- `registerAuraComponents()`
- `registerAuraMasterDetail()`
- `registerAuraTabs()`

Example:

```ts
import { registerAuraComponents } from "@aura/components";

registerAuraComponents();
```

For no-build browser usage in this repo, load the root `/aura-components.js`
entrypoint instead.
